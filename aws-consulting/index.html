About Amazon Web Services and SoftwareMill

AWS is maintained by Amazon, providing a versatile and powerful cloud computing service for web apps.

 

AWS is nowadays virtually everywhere - it powers NASA, Nokia, Lionsgate, Pfizer and more.

 

In addition to AWS, we develop our projects using Bootzooka, SoftwareMill-made and maintained bootstrap that saves you the time needed to build the basic functionality of your software.

Bootzooka, our Scala bootstrap

Bootzooka allows us to start your project quickly, and deliver you value as fast as possible. Here's Bootzooka's tech stack:


• Scalatra for creating REST APIs
• AngularJS, stylus, Twitter Bootstrap for the frontend
• joint SBT/Grunt build process
• Scala as the main backend development language
• Akka for asynchronous and background processing
• MongoDB as a persistent store

We are also running a series of smaller open source projects: Veripacks, ElasticMQ and MacWire. All written in and leveraging the huge possibilities of Scala.
