Scala development

Scala projects by SoftwareMill

Scala scales easily and is perfect for big projects. It’s really productive - developers can reduce the size of the source code by a factor of at least 2 or 3 compared to Java. On top of that, SoftwareMill is Typesafe's (Scala creators) consulting partner.

 

Case study: Yap.TV 

We developed the backend system for Yap.TV, a social TV app. The backend spans multiple servers, and is implemented using Akka and Spray. Used since the inception of the project, Scala has proved a powerful and concise language allowing us to create readable code and deliver features quickly at the same time. 

 

Case study: transforming data with Hadoop

We developed the backend for an analysis application that aggregates information regarding various customer behaviour patterns. Using this data, merchants are able to plan their marketing campaigns. To achieve the goals of this project, we used the Scala language together with Apache Hadoop and Apache Pig to transform the customer data, and Spray to serve the results.

About Scala and SoftwareMill

Scala is getting bigger every year, powering Twitter, The Guardian, Wallmart, Sony and Huffington Post. It's the next Java. It's perfect for big systems that need to scale.

 

Over and above being an expressive, scalable and fun language to write in, Scala has a rich ecosystem that is one of the main things which make it so great. The Lightbend stack and frameworks like Akka, Spray, Scalding or Scalatra make it much simpler and more efficient to achieve our daily development tasks.

 

We develop our projects using Bootzooka, SoftwareMill-made and maintained bootstrap that saves you the time needed to build the basic functionality of your software.

Bootzooka, our Scala bootstrap

Bootzooka allows us to start your project quickly, and deliver you value as fast as possible. Here's Bootzooka's tech stack:


• Scalatra for creating REST APIs
• AngularJS, stylus, Twitter Bootstrap for the frontend
• joint SBT/Grunt build process
• Scala as the main backend development language
• Akka for asynchronous and background processing
• MongoDB as a persistent store

We are also running a series of smaller open source projects: Veripacks, ElasticMQ and MacWire. All written in and leveraging the huge possibilities of Scala.
